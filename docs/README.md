### [chapt3.md](./chapt3.md) - 直方图与灰度变换
**涵盖内容**：
- ✅ 直方图计算与绘制
- ✅ 线性灰度变换（对比度/亮度调节）
- ✅ 对数变换（增强暗部细节）
- ✅ Gamma 变换（非线性调节）
- ✅ Otsu 自动阈值二值化
- ✅ 分段线性变换
- ✅ 直方图均衡化

**核心公式**：
```
线性变换: s = α·r + β
对数变换: s = c·log(1+r)
Gamma变换: s = 255·(r/255)^γ
Otsu阈值: T* = argmax σ²ᴮ(T)
均衡化:   s = (CDF(r) - CDFₘᵢₙ)/(total - CDFₘᵢₙ) × 255
```

**适合读者**：想要增强图像、调整对比度、自动分割的开发者

### [chapt4.md](./chapt4.md) - 几何变换
**涵盖内容**：
- ✅ 图像平移变换
- ✅ 图像翻转（水平/垂直/双向）
- ✅ 图像转置
- ✅ 图像缩放（最近邻/双线性/三次插值）
- ✅ 图像旋转（最近邻/双线性/三次插值）
- ✅ Catmull-Rom 三次插值详解

**核心公式**：
```
旋转矩阵: R(θ) = [cos(θ) -sin(θ)]
                 [sin(θ)  cos(θ)]

双线性插值: P = (1-dx)(1-dy)P₀₀ + dx(1-dy)P₁₀ 
              + (1-dx)dy·P₀₁ + dx·dy·P₁₁

三次权重: w(t) = (a+2)|t|³ - (a+3)|t|² + 1  (|t|≤1, a=-0.5)
```

**适合读者**：需要进行图像配准、矫正、数据增广的开发者

### [chapt5.md](./chapt5.md) - 图像滤波与边缘检测
**涵盖内容**：
- ✅ 卷积基础理论
- ✅ 均值滤波（平滑去噪）
- ✅ 高斯滤波（保边缘去噪）
- ✅ 中值滤波（去椒盐噪声）
- ✅ Robert 算子（2×2 边缘检测）
- ✅ Sobel 算子（3×3 边缘检测）
- ✅ Laplacian 算子（二阶边缘检测）

**核心公式**：
```
卷积: O(y,x) = Σᵢⱼ I(y+j,x+i)·K(j,i)

Sobel梯度: M = √(Gₓ² + Gᵧ²)

Laplacian: ∇²I = ∂²I/∂x² + ∂²I/∂y²

中值滤波: O(y,x) = median{I(y+j,x+i)}
```

**适合读者**：需要图像降噪、边缘提取、特征检测的开发者

### [others.md](./others.md) - 单目与双目视觉测量
**涵盖内容**：

#### 单目视觉测量

- ✅ 内参矩阵与坐标转换
- ✅ 图像到世界坐标的逆投影
- ✅ 3D 距离、周长、面积计算

#### 双目视觉测量（八点法）
- ✅ 对极几何基础
- ✅ 基础矩阵理论
- ✅ 八点法算法推导
- ✅ 点归一化技术
- ✅ SVD 求解与秩约束
- ✅ 对极线绘制与验证

**核心公式**：
```
单目投影: [u,v,1]ᵀ = K[X/Z, Y/Z, 1]ᵀ

逆投影:   X = (u-cₓ)·Z/fₓ
         Y = (v-cᵧ)·Z/fᵧ

对极约束: xᵣᵀ·F·xₗ = 0

归一化:   T = [s    0   -s·x̄]
             [0    s   -s·ȳ]
             [0    0    1  ]
         其中 s = √2/dₐᵥᵧ
```

**适合读者**：进行 3D 测量、深度估计、立体视觉的开发者

## 📊 算法复杂度对比

| 算法 | 时间复杂度 | 空间复杂度 | 质量 | 文档 |
|------|------------|------------|------|------|
| 最近邻插值 | O(WH) | O(1) | 低 | chapt4.md |
| 双线性插值 | O(4WH) | O(1) | 中 | chapt4.md |
| 三次插值 | O(16WH) | O(1) | 高 | chapt4.md |
| 均值滤波 | O(9WH) | O(WH) | 中 | chapt5.md |
| 中值滤波 | O(9log9·WH) | O(9) | 高（椒盐噪声） | chapt5.md |
| Sobel边缘 | O(9WH) | O(WH) | 好 | chapt5.md |
| Otsu阈值 | O(256n) | O(256) | 自动 | chapt3.md |
| 直方图均衡 | O(256n) | O(256) | 好 | chapt3.md |
| 八点法 | O(n) | O(9n) | 依赖点数 | others.md |

注：W=宽度，H=高度，n=像素总数

## 🎯 实用技巧

### 1. 噪声处理建议
- **高斯噪声** → 高斯滤波
- **椒盐噪声** → 中值滤波
- **保留边缘** → 高斯滤波 > 均值滤波

### 2. 边缘检测建议
- **速度优先** → Robert 算子
- **质量优先** → Sobel 算子
- **精确定位** → Laplacian（需先降噪）

### 3. 图像缩放建议
- **快速预览** → 最近邻
- **一般应用** → 双线性
- **高质量** → 三次插值

### 4. 灰度变换建议
- **暗图增亮** → 对数变换 或 Gamma<1
- **亮图压暗** → Gamma>1
- **整体增强** → 直方图均衡化
- **调对比度** → 线性变换
